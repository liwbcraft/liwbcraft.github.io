## 📝 简介 | About

一个简洁的个人主页，展示我的技术栈、项目和兴趣爱好。支持明暗主题跟随系统，中英文双语切换。

A clean personal website showcasing my tech stack, projects, and interests. Supports system theme following and bilingual switching (Chinese/English).

## ✨ 特点 | Features

- 🎨 毛玻璃效果 + 现代化设计
- 🌙 明暗主题自动跟随系统
- 🌍 中英文双语切换
- 📱 响应式布局，支持移动端
- 🧩 展示 Obsidian 插件、Java 开发、AI 实验等项目

## 🛠️ 技术栈 | Tech Stack

- HTML5 / CSS3
- 原生 JavaScript（无框架依赖）
- Flexbox / Grid 布局
- CSS 变量（支持明暗主题）
